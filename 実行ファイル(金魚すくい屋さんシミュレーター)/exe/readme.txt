◆作品について◆

◇使用言語 : C++
◇SDK      : DirectX
◇開発期間 : 2025/12～2026/01
◇ジャンル : 見下ろし箱庭
◆動作確認済み環境◆
Windows11

◆操作方法◆
<<キーボード>>
A : 左に移動
D : 右に移動
RSHIFT : 釣り糸を途中で回収する
ENTER : 釣り糸を出す・アイテムをつかむ
SPACE : 次に進む
カーソル：水面をタッチする(真ん中にいるとき)
ESC : 終了

<<コントローラー>>※XINPUTのコントローラーで動作します。
左十字キー : 左に移動
右十字キー : 右に移動
Bボタン : 釣り糸を途中で回収する
Xボタン : 釣り糸を出す・アイテムをつかむ
START : 次に進む
左スティック+Aボタン：水面をタッチする(真ん中にいるとき)

◆ゲームの進み方◆
お客さんの対応をして評価とお金を稼ぐ
制限時間2分

◆工夫した点・苦労した点
メッシュを使って波紋の表現を出せるように工夫しました。(mesh.cpp・wave.cpp)
お客さんの行動の流れをステートパターンを利用して制御しました。(customerState.cpp)
チュートリアルを段階的にするためスペックパターンを利用しました。(controllTutorial.cpp・controllTutorial.h)
水面をきれいに再現するためHLSL言語を使ったシェーダに挑戦しました。(data/SHADER/water.hlsl)

◆反省点
操作のチュートリアルに満足してゲーム内容の説明がおろそかになってしまった。
アイテムを取るタイミングの表現がユーザーにとってわかりずらいままになってしまった。
